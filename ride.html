RIDE is a smartphone application operated by Stardibs.corp that offers cab-hailing and booking services.
About Ride:
BOOKING- commuters can find ride easily, and can book a driver for future transactions
PERSONALIZED LOCATION- users can select their desired pick up and drop off location. It user different sources like Wi-Fi, GPS, cellular networks, and others to access location.
MINIMIZE REPEAT BOOKING- the system will automatically find a nearest driver so users donâ€™t need to restart the booking process
COMMUNICATION- users can view and have the contact details of the driver together with the plate number.
TRANSACTION HISTORY- both commuters and drivers have transaction history
RATINGS AND EVALUATION- the user can evaluate the driver based on the experience rendered. Other commuters can see the ratings and evaluation of the driver given by other users.
ABSOLUTELY FREE- no additional fees on both drivers and commuters.
