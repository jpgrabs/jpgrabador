Type: Thesis Project

Back to Home

PEP (Private Eye Patroller) PEP is integrated with web server that can manage all the crime reports coming from the mobile user.

FEATURES

Login Page: Design only for admin. A fix username and password from the database. That can access and manage all of the report coming from the mobile user.

Main Page: Where you can see all of the latest reports coming from the PEP mobile user. Where you can see buttons that will led you to the different pages to manage the system.

Locate Page: The main function of the system where you can locate the address of the crime and displays the important data send by mobile users. Like date, time, crime, description.

Respond Page: The page where the admin will choose a team officer to come in the crime scene. And file again with case id, date and time.

Invalid Report Page: Page that can filter prank reports from unreliable mobile user. Because of the void data that their sent.
