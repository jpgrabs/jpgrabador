Mpos

"Open Parking Reservation Mobile Application"


Service: Mobile Development
Client: Company: (Capitalwise Corp.)
Type: Project Based
Development: November 2016 - February 2017
Back to Home


This is a mobile street parking application,it is also stand alone mobile app where the database of user and parking reservation is save on sql lite database. The application are also inclusive on private company and also can be used on the specific place and currently que list application. One of the features of this application is to import the data into excel, the receipt can be printed via bluetooth printer with the bluetooth integration. It has security features when the application is installed on the smart phone, it will open automically at the open of the application and it can't be uninstalled unless they formatted via bios. It can also have a user level integration, the admin that can controlled and manage everything, the technical support (it support) where they can monitor and navigate if theirs an issue, the Supervisor of the area that manage the employee and give the application when the parking officer are time in and the parking officer if the front row to assist the customer.
