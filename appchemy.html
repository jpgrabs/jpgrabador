AppChemy

"Game Application"


Service: Android Mobile Development
Client: Students
Type: School Project
Development: September 2016 - November 2016

Back to Home




AppChemy is an android-based game application aims to help the students to learn how to bond chemicals correctly using Lewis Electron Dot System. The concept behind this application is a traditional card game combined with the concept of balancing chemicals using a click motion event that will trigger the move on a particular question.
I been part to be one of the assist / adviser on this group and complete the and upload the game application on my playstore account.
